Local Development
1. Install oracle database on your machine if not already installed.
2. Unzip the contents of zip file in a local folder
2. Locate .env file in the root directory and provide values for the keys
4. Run PL/SQL commands in schema.sql file to create tables, procedures and functions
5. Download sample data from https://github.com/BestBuyAPIs/open-data-set
6. Run Main.js file.
